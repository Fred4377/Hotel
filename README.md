# Grand Horizon Hotel 🏨

Hello! 👋 This is the Grand Horizon Hotel landing page, a luxury hotel website design I built to practice advanced CSS layout techniques, glassmorphic UI overlay panels, cinematic header slideshows, and typewriter animations.

## 📸 Preview
![Grand Horizon Hotel Preview](screenshots/preview.png)

## Why I built this
I wanted to challenge myself to build a truly premium-looking layout that screams "luxury." Most hotel sites are a bit outdated, so I went for a dark aesthetic combined with clean gold/amber accents, large elegant typography, and translucent glassmorphism cards. I spent a lot of time getting the hover transitions and scroll reveals to feel fluid and premium!

## Features
- Cinematic full-screen hero slideshow with typewriter animations
- Luxury room showcase section with detailed pricing and amenities
- Translucent glassmorphism cards and sticky blurred navbar
- Responsive contact form and styled location placeholder
- Smooth scroll animations using a lightweight JavaScript engine

## Tech Stack
- **Frontend:** HTML5, CSS3 (Advanced Flexbox/Grid, transitions)
- **Scripting:** Vanilla JavaScript
