# 🏨 Grand Horizon Hotel - Luxury Emerald & Gold Safari Lodge

![Preview](screenshots/preview.png)

A premium luxury resort landing page themed with a rich **Emerald Green & Warm Gold** aesthetic, presenting boutique lodgings.

## ✨ Humanized & Localized Features
- 🦁 **Kenyan Lodge Listings** — Room options customized to premium local destinations (e.g. Mara River Luxury Tent, Diani Beach Ocean Suite, Naivasha Lake Lodge) in KSh.
- 💳 **Lipa na M-Pesa Deposit Flow** — A simulated reservation deposit checkout modal utilizing Safaricom M-Pesa STK inputs.
- 🌟 **Guest Review Cards** — Reviews with custom rating displays highlighting local travel experiences.

## 🛠️ Tech Stack
- **Frontend:** HTML5, Vanilla CSS, JavaScript
- **Typography:** Cormorant Garamond & Lato
